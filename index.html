Proyectos Destacados

Pokedex

Aplicación Web Interactiva y Consumo de APIs desarrollada con React y TypeScript. Diseñada con un fuerte enfoque en UI/UX, implementa animaciones fluidas, scroll infinito y cruce de datos en tiempo real entre múltiples APIs.

MundoHardware

Tienda online con catálogo interactivo de componentes informáticos. Diseñado para ofrecer una experiencia de usuario rápida y moderna.

Modelo Acueducto 3D

Landing page interactiva en 3D. El proyecto destaca un modelo estilo Low Poly del Acueducto de los Milagros de Mérida, integrado en un entorno web moderno utilizando React Three Fiber, Tailwind CSS y componentes de Shadcn UI.

Aventura en el reino JS

Un juego de aventuras con sistema de escenas, donde el jugador compra objetos en un mercado, se enfrenta a batallas automáticas contra enemigos para ganar puntos

API Gasolineras

Aplicación web que permite consultar las gasolineras de España usando la API oficial del Estado. El usuario puede filtrar por provincia, municipio, tipo de combustible y si la estación está abierta

Figma

Guía de estilos y prototipo de portfolio.

Consultoría web

Landing page animada que presente los servicios de una consultoría de desarrollo web, mediante el uso de GSAP.

Tablero Kanban

Tablero Kanban web interactivo hecho con HTML, CSS y JavaScript. Permite agregar, mover y eliminar tarjetas en columnas Todo, Doing y Done

Panel de Usuario

Aplicación web que permite a los usuarios registrarse, iniciar sesión y acceder a un panel personalizado. Incluye validación en tiempo real de formularios, gestión de modo claro/oscuro, y la permanencia de datos mediante LocalStorage y cookies
